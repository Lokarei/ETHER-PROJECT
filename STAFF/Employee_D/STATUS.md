5.2 Замыкание
Уравнение 1 — КУБК-невидимость (§4.5 CANON; §6.2 базы A):

n_ch · |v|² = n_a · v₀²
Поток импульса через ГП от канального газа = поток через любую площадку в СЭ. СЭ «не видит» границу вихря.

Уравнение 2 — порог устойчивости (§4.3 CANON; §4.2 базы A):

|v_n| ≥ 2·|v|
Рабочее значение: |v_n| = 2·|v| (нижний порог).

Результат:

|v|  = |v_n| / 2
n_ch = n_a · v₀² / |v|² = n_a · 4·v₀² / |v_n|²
При |v_n| = 3·v₀: |v| = 1.5·v₀, n_ch = 0.444·n_a.

5.3 КУБК-инвариант дрейфа
V_∥ не зависит от |v_n|. При замыкании n_ch·|v|² = n_a·v₀² = const произведение n_ch·v_chaos² фиксировано. Геометрический вклад v_n нормируется на v_n² через cos²α и сокращается.

Проверено численно:

|v_n| = 2·v₀  →  V_∥ = 5.861·10⁻⁶
|v_n| = 3·v₀  →  V_∥ = 5.861·10⁻⁶
|v_n| = 5·v₀  →  V_∥ = 5.861·10⁻⁶
Физический смысл: скорость дрейфа фотона определяется ТОЛЬКО фундаментальными константами СЭ (n_a, v₀, d_A) и геометрией ЭВО (A, k_z, r), не зависит от деталей канального потока.
