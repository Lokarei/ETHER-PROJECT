## 1. Канон-опора (CANON 21.03)

### 1.1 Фундаментальные меры (§2.2)
d_A — длина; v₀ = ⟨|v_a|⟩_СЭ — скорость; τ₀ = d_A/v₀ — время.

### 1.2 Константы СЭ (§2.1, §2.3)
n_a = 16/(27√2·d_A³); ε ≈ 0.219; v_max = √2·v₀; v_s = 3√2·v₀; p_СЭ = m_a·v₀²/(4d_A²).

### 1.3 Двухфазность (§4.1–4.3)
Хаос (⟨l⟩=d_A/2, псевдожидкость) ≠ Канал (⟨l⟩≫d_A, газ). Запрет смешения.

Двухкомпонентная скорость: v_вэ = v_n + v. Порог устойчивости: |v_n| > 2|v|.

### 1.4 КУБК = невидимость (§4.5)
Поток импульса через ГП = через любую площадку в СЭ. Разделение ролей: v_n → ГДЕ (геометрия, χ); v → СКОЛЬКО (КУБК-обмен).

### 1.5 Волна шнека (§6)
Вдавливание ГП внутрь канала давлением СЭ. Запрет: «канал выпячивает наружу».

### 1.6 Механизм дрейфа (§8.3)
```
P_СЭ → вдавливание ГП → смещение мест столкновений
  → χ-нечётное θ-смещение (h_A, χ)
  → Ω(χ) → V_∥(χ·s_sh)
```
Δp — сопутствующий эффект. Хаос не вносит θ-момент.

### 1.7 Трёхканальная геометрия (§14)
S₁:S₂:S₃ = 1:2:3; m_A = 6:3:2; инвариант S_k·m_Ak = const = 6.
